# 🚀 TaskFlow Todo

TaskFlow Todo is a clean and user-friendly React application designed to help you manage your daily tasks efficiently.

---

## ✨ Features

- ➕ Add new tasks
- ❌ Delete tasks
- ✅ Mark tasks as completed
- 🧹 Clear completed tasks (optional)
- 🎯 Simple and intuitive UI

---

## 🛠️ Tech Stack

- React
- JavaScript (JSX)
- CSS

---

## 📦 Installation

Follow these steps to run the project locally:

```bash
git clone https://github.com/UmangBheda/taskflow-todo.git
cd taskflow-todo
npm install
npm run dev
